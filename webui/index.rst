WebUI
*****

This manual is meant for app developers who want to build a Web-based User Interface (in short, WebUI) for their AIMMS applications. 
To a smaller extent, this manual may also useful as reference for the end users of such AIMMS WebUI apps, for example when using configuration options available to them when running the applications on the AIMMS PRO platform.
